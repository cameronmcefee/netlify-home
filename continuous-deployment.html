Continuous Deployment

Automating your builds and deploys with Netlify

Netlify makes it easy to set up a completely automated flow, that'll build and deploy your static site whenever something changes.

Tell Netlify where your git repository lives, whether it's on Github, BitBucket, GitLab or your own self-hosted repository.
Tell Netlify your build command, whether it's "jekyll build", "ember build", "roots compile" or "gulp". Netlify will run it on every push to Git
Tell Netlify what folder to deploy. Once the build command is done, Netlify will do an atomic deploy with instant cache invalidation to a super fast CDN

Continuous deployment makes it easy to run statically generated sites with up-to date content, or quickly iterate on the front-end for your single page app.
